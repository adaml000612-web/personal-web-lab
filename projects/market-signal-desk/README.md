# 前哨 · 投资情报雷达

一个在 Chrome 中使用的个人投资信息工作台。第一版覆盖 A 股、港股和美股，
关注英伟达、SpaceX、特斯拉、谷歌、腾讯、中际旭创，以及纳斯达克、标普 500、
上证指数。

## 本地运行

需要 Node.js 22.13 或更高版本。

```bash
npm install
npm run dev
```

然后在 Chrome 打开终端显示的本地地址（通常为 `http://localhost:3000`）。

## 1.0.0 能力

- 腾讯财经延迟行情（只展示成功取得的数据，不填充虚构价格）
- Nasdaq RSS、SEC EDGAR、腾讯投资者关系与 A 股公告索引
- P1 公司/官方、P2 板块/产业链、P3 指数、P4 海外映射四级排序
- 新闻收藏与已读状态保存在浏览器本机
- 巨潮资讯、港交所披露易、SEC、上交所官方入口

版本历史见 [CHANGELOG.md](CHANGELOG.md)。每次正式迭代都有独立 Git 提交和标签。

SpaceX 作为未上市公司的新闻观察对象，不展示股票价格。长鑫存储待正式上市并
确认交易代码后接入。

> 本工具仅用于信息整理，不构成投资建议。
